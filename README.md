Quick start
===========

```
$ sudo pip3 install virtualenv
$ virtualenv-3.4 ~/virtualenv/akamai
$ . ~/virtualenv/akamai/bin/activate

(akamai) $ python setup.py develop

# set the following environment variables
# - AK_BASE_URL
# - AK_CLIENT_TOKEN
# - AK_CLIENT_SECRET
# - AK_ACCESS_TOKEN

(akamai) $ akamai_cdn_report
```
